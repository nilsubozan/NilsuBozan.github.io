Top 50 YouTube Titans: Insights & In-Depth Analysis by Using Youtube API

The Top 50 Most Subscribed YouTube Channels were analyzed, with a detailed focus on the top 3 channels. Data was extracted directly from YouTube using the YouTube API, capturing key metrics such as views, likes, comments, and upload frequencies. Python was employed for data cleaning, ensuring the data was properly structured and ready for analysis. The core analysis was conducted using SQL, where trends, metrics, and insights were explored. Finally, the findings were visualized in Tableau, with interactive dashboards created to provide both a general overview and in-depth insights into the top YouTube channels.

US Consumer Price Index Analysis | January 2020 to August 2024

This project provides an interactive visualization of the US Consumer Price Index (CPI) from January 2020 to August 2024, focusing on various economic categories like Housing, Food, and Transportation. Data was extracted from the Bureau of Labor Statistics (BLS) and preprocessed using SQL to transform and clean the information. The data was unpivoted to create a consolidated CPI table, ensuring a comprehensive analysis across different time periods and categories. Tableau was utilized to design dashboards that reveal trends and insights into how consumer prices have shifted over time. The analysis offers valuable insights into inflation patterns, helping to understand the economic factors affecting the cost of living in the US.

COVID-19 Trends Analysis: Insights from 2020 to 2021

This project provides an analysis of global COVID-19 trends and vaccination efforts from January 2020 to August 2024, using data sourced from Our World in Data. SQL was employed for in-depth analysis and data cleaning, transforming the raw data into structured formats suitable for exploration. Key metrics analyzed include total cases, total deaths, death rates, infection rates, and vaccination progress across different countries and continents. The findings were then visualized in Tableau, offering an interactive platform for users to explore the evolution of the pandemic and its impact over time.
